Networking Concepts Demonstrated

DNS

Domain Name System (DNS) resolves our domain name into an IP address using recursive and authoritative name servers.

IP Addressing

The domain supports both IPv4 (A record) and IPv6 (AAAA record), demonstrating dual-stack deployment.

Protocols

The website uses HTTPS (HTTP over TLS), ensuring encrypted communication between client and server.

Security

HTTPS enforcement ensures secure data transmission. Security headers and TLS certificates protect against attacks.
